# Pinstagram

Instagram clone with Express, Prisma, React and React Native

## User Stories

- [x] Create account
- [x] Request Secret
- [x] Confirm Secret (Login)
- [x] Like / Unlike a Post
- [x] Comment on a Post
- [x] Search by user
- [x] Search by location
- [x] Follow User
- [x] Unfollow User
- [x] Edit my profile
- [x] See user profile
- [x] See my profile
- [x] See the full Post
- [x] Upload a Post
- [x] Edit the Post (Delete)
- [x] See the feed
- [x] Send private message
- [x] See Rooms
- [x] See Room
- [x] receive message (Realtime)
